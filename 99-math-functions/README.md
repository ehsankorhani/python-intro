# Mathematical methods

**abs()**: absolute value
```bash
>>> abs(-1.5)
1.5
```

**sum()**: sum of all numbers is a tuple.
```bash
>>> myTuple = (1, 3, 5, 7, 11)
>>> sum(myTuple)
27
```

**min()** and **max()**
```bash
>>> max(10, 18, 8)
18

>>> min(10, 18, 8)
8
```

**pow()**: returns the value of first number to power of second number.
```bash
>>> pow(2, 10)
1024
```

**round()**: rounds the specified number, with the specified number of decimals.
```bash
>>> round(18.652, 2)
18.65
```
